#  Insurance Claim Prediction App

## Project Overview
This project predicts **insurance claim amounts** based on personal and health-related features. Using historical insurance data, a machine learning model estimates expected claim payments to assist insurance companies and customers.

The project includes:
- Data preprocessing, cleaning, and exploration
- Model training using Linear Regression, Random Forest, and XGBoost
- Hyperparameter tuning for best performance
- Evaluation with MAE, RMSE, and RÂ² metrics
- Interactive **Streamlit** app for live predictions

---

## Key Features
- Predict claims using:
  - Age, Gender, BMI, Blood Pressure
  - Diabetic status, Smoking status, Number of children, Region
- Tuned **XGBoost** provides highest accuracy
- Visualize top features affecting claim amounts
- User-friendly interface with custom background
